# 🔍 Deepfake Fingerprint Forensic Tool
A cybersecurity tool designed to detect AI-generated media using Frequency Domain Analysis.

### 🚀 How to Run
1. `pip install -r requirements.txt`
2. `streamlit run app.py`

### 🛠️ The Tech
- **FFT Analysis:** Isolates invisible high-frequency artifacts.
- **Source Attribution:** Identifies fingerprints unique to GANs/Diffusion models.
